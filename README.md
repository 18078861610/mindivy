mindivy
=======

一个纯 JavaScript / HTML5 实现的思维导图编辑器 <br/>
如果问什么是思维导图的话。。一般来说用过 mindmanager 的朋友就知道。<br/>
如果不知道的话可以去搜索引擎找找介绍。算是比较有用的知识工具。


#### 开发进展

###### 20114.9.30

- 完成了基本的节点选择，文字编辑事件处理
- 完成了基本的节点布局算法
